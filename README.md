# riichi-mahjong

Riichi Mahjong hand calculation library for TypeScript.

## Features

- Hand calculation (han, fu, yaku, scores)
- Shanten calculation
- Agari (winning hand) detection
- TypeScript support with full type definitions

## Installation

```bash
npm install riichi-mahjong
```
